# Minimal Hybrid Plan (Gym + Bike)

## Goal
- Keep building muscle with shorter gym sessions
- Progress cycling slowly as a beginner
- Respect no-squat pain and shoulder-friendly pressing
- Keep recovery high so work performance stays good

## Weekly Structure
- Gym: 3 mandatory sessions + 1 optional light session, each 60-90 min (max 1.5h)
- Bike: 3 sessions (including Sunday social ride)
- Practical split with your Sunday ride: usually 50-60% gym, 40-50% bike
- Hard day cap: max 3 hard days/week total (gym + bike combined)

- Keep intensity easy (RPE 2-3/10), no fatigue, no sweating goal

## Weekly Schedule

### Monday - Gym Upper A (75-90 min)
- Machine chest press: 3x6-10
- Incline machine press: 2-3x6-10
- Chest-supported row: 3x6-10
- Neutral-grip pulldown: 3x6-10
- Lateral raise: 2-3x10-15
- Rope pressdown: 2-3x8-12
- Cable curl: 2-3x8-12
- Core finisher (8-10 min):
  - Cable crunch (heavy): 3x8-12
  - Hanging knee/leg raise (controlled): 2-3x8-15

### Tuesday - Gym Lower A (60-75 min)
- Leg press (pain-free ROM): 3x8-12
- Romanian deadlift (DB or Smith): 3x6-10
- Seated/lying leg curl: 3x8-12
- Leg extension: 2x10-15
- Calf raise: 3x8-12
- Optional adductor machine (light, pain-free): 1-2x12-15

### Wednesday - Bike Z2 (75-90 min)
- Full ride in Zone 2 (RPE 4-5/10)
- Smooth cadence 85-95 rpm

### Thursday - Gym Upper B (75-90 min)
- Machine chest press (different angle/grip from Monday): 3x6-10
- Seated cable row: 3x6-10
- Pulldown or assisted pull-up: 3x6-10
- Rear delt fly: 2-3x10-15
- Lateral raise: 2-3x10-15
- Overhead cable triceps extension: 2x8-12
- Incline DB or cable curl: 2x8-12
- Core finisher (8-10 min):
  - Machine crunch or weighted decline sit-up: 3x10-15
  - Cable side bend: 2-3x12-20/side

### Friday - Bike Tempo/Skill (60 min)
- Auto-regulated day:
  - If recovery is good: 10-15 min warm-up, 3x6 min Zone 3 (RPE 6-7/10), 4 min easy between, then 4x1 min high cadence
  - If recovery is poor/stress is high: full 45-60 min Zone 2 only

### Saturday - Optional Light Gym (30-45 min, only if fresh)
- Pick 2-4 exercises total, keep it easy/moderate:
  - Hip thrust or leg press (pain-free): 2x6-10
  - Leg curl: 2x8-12
  - One row or pulldown variation: 2x6-10
  - One shoulder/arm isolation: 1-2x10-15
  - Optional core: 1-2 sets machine crunch or cable crunch
- Stop with 2-3 reps in reserve (do not grind)

### Sunday - Social Peloton Ride (2-3h)
- Ride by feel with the group
- Try to keep most time in Zone 2, short Zone 3 pulls are fine
- Fuel during ride (see nutrition section)

## Gym Rules (Minimal but Effective)
- No painful squat pattern. Replace immediately if inner thigh pain appears.
- Use mostly machines and stable movements for joint comfort.
- Keep 1-2 RIR on compounds; 0-1 RIR on last isolation set.
- Rest times: 2-3 min compounds, 60-90 sec isolations.
- If short on time, cut the final isolation exercise first.
- For hard compounds, default rep range is 6-10.
- Core is mandatory on Monday + Thursday (short finishers only).
- Core work is hypertrophy style: loaded sets, not long-duration holds.

## Progression
- Double progression:
  - Stay in the listed rep range
  - Add reps week to week
  - When all sets hit top of range, add 2.5-5% load
- Core progression:
  - Add reps within range, then add load (do not increase time-based holds)
- Cycling progression:
  - Keep only Friday as moderate intensity
  - Add 5-10 min to Wednesday Z2 every 1-2 weeks if recovery is good

## Deload (Every 4th Week)
- Gym: cut sets by 40-50%
- Bike: cut volume by 30-40% and remove Friday Zone 3 blocks
- Core: keep same exercises but do only 1-2 easy sets each

## Recovery Guardrails (Priority)
- Max 3 hard days/week total.
- If sleep is poor for 2 nights in a row, reduce next session volume by ~30%.
- If legs feel heavy/stale, switch Friday to Zone 2 only.
- If performance drops for 2 gym sessions in a row, run an immediate easy week:
  - Gym sets -40%
  - Bike volume -30%
  - No Zone 3 work

## Nutrition Notes
- Keep lean gain surplus small: +150 to +250 kcal/day on average
- Protein: 1.8-2.2 g/kg/day
- For rides 90+ min: 30-60 g carbs/hour
- For Sunday 2-3h ride: 45-75 g carbs/hour + electrolytes

## Tracking
- Daily morning weight (use weekly average)
- Waist once weekly
- Gym logbook: reps/load/RIR
- Bike log: duration/HR/RPE

If fatigue rises, first reduce Friday bike intensity, then skip Saturday optional gym.
